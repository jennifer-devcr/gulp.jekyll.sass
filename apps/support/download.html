RIMES EXCEL ADD-IN

RIMES Excel Add-in allows customers to screen and aggregate multiple universes, perform time-series calculations and analysis, and create user-defined portfolios. It also helps with executing complex calculations and analytics, which are performed server side. Customers using the RIMES Excel Add-in can minimize the strain on Excel and maximize workbook efficiency.

Key Features

Report wizard provides a user-friendly interface to create personalized reports.

Interactive and editable web reports downloaded from online.rimes.com

Seamlessly integrated into Excel: toolbar automatically displays when there is RIMES content in the current spreadsheet.

RIMES CODE LOOKUP

The RIMES Code Lookup application is designed to help you build data request queries by providing you with a means to view, browse, search, and/or apply identifier codes for the following data request specifications:

  • Financial database sources (e.g. MSCI, BofA/Merrill, FTSE, S&P Dow Jones, etc.)
  • Company and index symbols (e.g. indices, companies, bonds, etc.)
  • Cross-referenced items (e.g. current, historical, portfolio)

Key Features

Matching sources appear as you type.

You can select any source, symbol or item and then copy/paste to your Excel spreadsheet.
